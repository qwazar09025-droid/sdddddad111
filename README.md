# Команда мечты — Django CRUD

Небольшое учебное CRUD-приложение на Django 5: управление списком участников
команды (имя, возраст) с AJAX-операциями без перезагрузки страницы.

## Что нового в этом дизайне

- 🎨 **Современная тема** на Tailwind CSS v4 с градиентами и стеклянным эффектом (`backdrop-filter`).
- 🌗 **Тёмный/светлый режим** с переключателем и сохранением в `localStorage`,
  без мигания при загрузке.
- 📇 **Карточный список** с аватарами-инициалами (градиент генерируется из имени).
- 🔍 **Живой поиск** по имени.
- 📊 **Сводная панель** (всего участников, средний/мин/макс возраст).
- 🔔 **Тост-уведомления** вместо `alert()`; **кастомное модальное окно**
  подтверждения вместо `confirm()`.
- ⌨️ Закрытие модалок по `Esc` и клику вне.
- 💵 Страница `finance.html` оформлена как скелет финансовой панели
  (KPI-карточки, плейсхолдер графика, лента операций) в той же стилистике.

Бэкенд (модели, формы, URL-ы, AJAX-контракт) сохранён без изменений —
обновился только слой представления и стили виджетов формы.

## Структура

```
Course_ThirdProject/
├── Course_ThirdProject/    # настройки проекта Django
├── crud/                   # приложение CRUD (models, views, forms, urls)
├── templates/              # index.html, finance.html
├── manage.py
└── requirements.txt
```

## Быстрый старт

```bash
cd Course_ThirdProject
python -m venv .venv
source .venv/bin/activate            # Windows: .venv\Scripts\activate
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```

Открыть в браузере: <http://127.0.0.1:8000/>
